### NERACOOS WAF Scripting Examples ###

For the NERACOOS THREDDS catalog at: <http://www.neracoos.org/thredds/UMO_SOS_historical_realtime_agg.xml>  
*should work for any TDS*

* `WAF_generator.py`: Example python script to harvest ISO metadata records from an ncISO enabled THREDDS server.
  Creates a WAF for the NERACOOS THREDDS catalog at: <http://www.neracoos.org/thredds/UMO_SOS_historical_realtime_agg.xml>

  Requires [thredds_crawler:]( https://github.com/asascience-open/thredds_crawler)
